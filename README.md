## ✨ Features

- **Interactive Animations:** Watch Gojo stretch, look around, jump, sleep, and even use his abilities.
- **AI Chat (Gemini Powered):** Talk to Gojo directly! Powered by Google's Gemini API, he responds with his characteristic personality.
- **Dynamic Stats:** Monitor his Mood, Energy, and Hunger. Keep him happy and energetic!
- **Context Menu Actions:** Right-click on him to feed him, pet him, tell him to sleep/wake, or trigger signature moves like *Domain Expansion* or *Infinity*.
- **Rizz Mode:** Enable it in the settings if you can handle it. 😉
- **Run Mode:** Make him run around your screen.
- **Clipboard Reacts:** He might just playfully comment on what you're copying.

## 🚀 Installation

It's extremely easy to get Gojo running on your Windows machine:

1. Head over to the [Releases page](https://github.com/Chaitanyahoon/gojo-desktop/releases).
2. Download the latest `Gojo-Pet-Setup-x.x.x.exe`.
3. Run the installer.
4. Enjoy! A shortcut will be automatically created on your Desktop and Start Menu.

## ⚙️ Configuration

Right-click Gojo and open **Settings** to customize his behavior:
- Set your **Gemini API Key** to enable the AI chat functionality.
- Adjust his walk speed and interaction frequency.
- Toggle features like *Always on Top*, *Start on Login*, and *Speech Bubbles*.

## 🛠️ Tech Stack

Built entirely with standard web technologies inside an Electron wrapper:
- **Electron** for native Windows capabilities and transparency.
- **HTML/CSS/JS** for the UI, canvas rendering, and physics.
- **Node.js** for file system handling and API communication.
